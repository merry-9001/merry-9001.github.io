git操作

2019-10-25

git 需要配置邮箱 账号

然后设置公钥（在github里面）

查看仓库
git remote

创建仓库
git remote add origin 地址

创建分支
git checkout -b "dev"

查看状态
git status

添加
git add .

提交（注释）
git commit -m "内容"

切换分支
git checkout master

合并分支
git merge dev --no-ff

退出注释
: q

查看日志
git log

上传至github
git push origin master

查看分支
git branch

删除
git branch -d dev

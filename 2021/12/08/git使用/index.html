git使用

  • 通用计算机知识

github 连接密钥

新建.ssh

进入然后 git bash
ssh-keygen -t rsa -C "739211562@qq.com"

设置

1.设置用户名
git config –global user.name ‘用户名’
2、设置用户邮箱
git config –global user.email ‘用户邮箱’
3、查看设置
git config –list

git push 时需要设置 authorize

他会跳出一个 github 的验证输入

config 配置指令

git config

config 配置有 system 级别 global（用户级别） 和 local（当前仓库）三个 设置先从 system-》global-》local 底层配置会覆盖顶层配置 分别使用–system/global/local 可以定位到配置文件
查看系统 config

git config –system –list

查看当前用户（global）配置

git config –global –list

查看当前仓库配置信息

git config –local –list

查看仓库

git remote

创建仓库

git remote add origin 地址

创建分支

git checkout -b “dev”

查看状态

git status

添加

git add .

提交（注释）

git commit -m “内容”

切换分支

git checkout master

合并分支

git merge dev –no-ff

退出注释

: q

查看日志

git log

上传至 github

git push origin master

查看分支

git branch

删除

git branch -d dev

git -f 可以强制执行

问题

git -am 与 git -m

知识

git push 的时候不区分大小写
所以改大小写不会修改远程代码

dll:动态链接库

有一些文件，他暴露了一些 api,你可以随时将这些 api 里面的东西拿出来用

BFF：backend for Frontend 中间层，在前端和后端之间夹了一层，这层为前端服务的

为什么需要 BFF

因为前端需要的数据和后端存储的数据不一致

事务：最小的，不可分割的任务

hash：hash 是一种算法，可以将任何长度的数据转换为定长的数据，常见的 hash 算法包括 MD5、SHA-1。

依赖倒置原则：当一些具体的实现可能会变动的时候就不要去依赖具体实现，而是直接依赖接口

写法疑问

class Chameleon {
  static colorChange(newColor) {
    this.newColor = newColor;
  }

  constructor({ newColor = "green" } = {}) {
    this.newColor = newColor;
  }
}

const freddie = new Chameleon();

{ newColor = “green” } = {} 这个写法是什么意思？

相当于是给了一个默认的对象值，如果没有传对象参数那么就使用前面的默认值，如果传递了则会覆盖前面的对象值（后面这样写应该是固定语法）

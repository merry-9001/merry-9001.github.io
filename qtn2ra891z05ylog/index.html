grid（绘图网格）

可以在网格上绘制折线图，柱状图，散点图（气泡图）。

直角坐标系内绘图网格，单个 grid 内最多可以放置上下两个 X 轴，左右两个 Y 轴

1.图表的背景显示网格，以及背景颜色等

2.确定图表的位置

grid:{
  // show:true,
  left:"20%",
  width:"100%",
  containLabel:false,
  backgroundColor:'gray',
  borderColor:'#f00'
},
